recent projects



Discover Weekly Saver

A Flask web app that lets users save their Discover Weekly playlist to their own Spotify account before it disappears at the end of the week. It uses the Spotipy library and authenticates using OAuth 2.0

HandTrack Mouse

A virtual mouse built using OpenCV and Mediapipe. It uses hand tracking through a webcam. It enables mouse movement, scrolling, and gesture-based interaction such as left and right clicks, all controlled by specific finger positions and motions.

Starry Night Cellular Automaton

A simulation of a pseudo-cellular automaton where each unit behaves as an FSM and follows a simple state cycle: inactive, swirl, fade, repeat. The result is an organic motion loosely modeled on Van Gogh’s The Starry Night.

Autonomous EV3 Car with Accessibility Ramp

An autonomous LEGO EV3 vehicle programmed in MATLAB to navigate a maze while obeying simulated traffic rules. The car uses color and motion sensors to detect lanes, ramps, and signals for real-time decision-making and path correction.
